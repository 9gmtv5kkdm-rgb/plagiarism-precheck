## English

Plagiarism Pre-Check & Reduction Methodology — not a detection tool, but a **comprehensive guide to understanding plagiarism check rules and structural reduction strategies**.

### Coverage
- **CNKI/Turnitin principles**: 13-character similarity matching, chapter-level checking, citation exclusion
- **6-tier rewriting strategy pyramid**: from comprehension-based reorganization to synonym replacement (last resort)
- **Citation boundaries**: what to cite, what not to cite, self-citation guidelines
- **AI + plagiarism dual strategy**: AI-generated text detection characteristics, dual-check workflow
- **Pre-submission checklist**: 6-item checklist + multi-skill workflow integration

### Relationship with existing skills
- `plagiarism-checker`: Detection tool → This skill provides methodology
- `paper-plagiarism-reducer`: Replacement tool → This skill provides structural strategies
